### 描述

- 该接口提供版本：v9.9.9+。
- 该接口所需权限：IaaS资源操作。
- 该接口功能描述：批量关闭虚拟机,异步接口&支持任务管理。（需注意采用关机方式为强制关机）

### URL

POST /api/v1/cloud/cvms/batch/stop_async

### 输入参数

| 参数名称 | 参数类型         | 必选  | 描述       |
|------|--------------|-----|----------|
| ids  | string array | 是   | 虚拟机的ID列表 |

### 调用示例

```json
{
  "ids": [
    "00000001",
    "00000002"
  ]
}
```

### 响应示例

```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "task_management_id": "xxxxxx"
  }
}
```

### 响应参数说明

| 参数名称    | 参数类型   | 描述   |
|---------|--------|------|
| code    | int32  | 状态码  |
| message | string | 请求信息 |
| data    | object | 响应数据 |


#### data参数说明

| 参数名称               | 参数类型   | 描述     |
|--------------------|--------|--------|
| task_management_id | string | 任务管理id |
