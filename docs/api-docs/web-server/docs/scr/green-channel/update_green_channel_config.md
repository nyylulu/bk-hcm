### 描述

- 该接口提供版本：v1.7.0.0+。
- 该接口所需权限：平台管理-小额绿通。
- 该接口功能描述：更新小额绿通配置。

### URL

PATCH /api/v1/woa/green_channels/configs

### 输入参数

| 参数名称                 | 参数类型 | 必选 | 描述                                   |
|----------------------|------|----|--------------------------------------|
| biz_quota            | int  | 否  | 业务申请小额绿通的每周额度，不传此字段时，将不会更新此字段对应配置    |
| ieg_quota            | int  | 否  | 整个ieg申请小额绿通的每周额度，不传此字段时，将不会更新此字段对应配置 |
| audit_threshold | int  | 否  | 小额绿通自动审批核数上限，不传此字段时，将不会更新此字段对应配置     |

### 调用示例

```json
{
  "biz_quota": 500,
  "ieg_quota": 50000,
  "audit_threshold": 1000
}
```

### 响应示例

```json
{
  "code": 0,
  "message": "ok"
}
```

### 响应参数说明

| 参数名称    | 参数类型   | 描述   |
|---------|--------|------|
| code    | int    | 状态码  |
| message | string | 请求信息 |
