### 描述

- 该接口提供版本：v1.7.4+。
- 该接口所需权限：无。
- 该接口功能描述：查询需求类型对应的OBS项目类型列表。

### URL

POST /api/v1/woa/meta/requirement/obs_project/list

### 输入参数

无

### 响应示例

```json
{
  "code": 0,
  "message": "",
  "data": {
      "1": "常规项目",
      "2": "2025春节保障",
      "3": "2025机房裁撤",
      "6": "滚服项目",
      "7": "小额绿通"
  }
}
```

### 响应参数说明

| 参数名称    | 参数类型   | 描述   |
|---------|--------|------|
| code    | int    | 状态码  |
| message | string | 请求信息 |
| data    | object array | 响应数据 |
