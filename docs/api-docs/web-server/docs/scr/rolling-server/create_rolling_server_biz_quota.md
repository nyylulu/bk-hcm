### 描述

- 该接口提供版本：v1.6.11+。
- 该接口所需权限：平台管理-滚服管理。
- 该接口功能描述：创建业务滚服基础额度。无法为已有基础额度的业务再次创建

### URL

POST /api/v1/woa/rolling_servers/biz_quotas/batch/create

### 输入参数

| 参数名称        | 参数类型      | 必选 | 描述                           |
|-------------|-----------|----|------------------------------|
| bk_biz_ids  | int array | 否  | 业务ID列表，最大100，不传时为全部业务创建基础额度  |
| quota_month | string    | 是  | 额度所属月份，格式：YYYY-MM，例如：2024-09 |
| quota       | int       | 是  | 业务滚服基础额度（大于等于0）              |

### 调用示例

```json
{
  "bk_biz_ids": [
    639
  ],
  "quota_month": "2024-09",
  "quota": 10000
}
```

### 响应示例

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "ids": [
      "00000001"
    ]
  }
}
```

### 响应参数说明

| 参数名称    | 参数类型   | 描述                        |
|---------|--------|---------------------------|
| code    | int    | 错误编码。 0表示success，>0表示失败错误 |
| message | string | 请求失败返回的错误信息               |
| data	   | object | 响应数据                      |

#### data

| 参数名称 | 参数类型         | 描述       |
|------|--------------|----------|
| ids  | string array | 滚服额度配置ID |
