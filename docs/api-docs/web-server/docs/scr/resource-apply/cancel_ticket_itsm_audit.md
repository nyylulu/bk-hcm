### 描述

- 该接口提供版本：v9.9.9+。
- 该接口所需权限：平台管理-主机申领。
- 该接口功能描述：取消主机申请的当前ITSM审批单据。

### URL

POST /api/v1/woa/task/apply/ticket/itsm_audit/cancel

### 输入参数

| 参数名称     | 参数类型 | 必选 | 描述      |
|----------|------|----|---------|
| order_id | int  | 是  | HCM订单ID |

### 调用示例

```json
{
  "order_id": 1234
}
```

### 响应示例

```json
{
  "code": 0,
  "message": ""
}
```

### 响应参数说明

| 参数名称    | 参数类型   | 描述   |
|---------|--------|------|
| code    | int32  | 状态码  |
| message | string | 请求信息 |
