### 描述

- 该接口提供版本：v1.6.1+。
- 该接口所需权限：无。
- 该接口功能描述：资源回收主机列表查询。

### URL

POST /api/v1/woa/task/find/recycle/host

### 输入参数

| 参数名称   | 参数类型   | 必选 | 描述          |
|-----------|----------|------|--------------|
| order_id	| int	   | 是   | 资源回收单据ID |
| status    | string   | 否   | 子任务状态。若为空，则返回所有主机列表。"RUNNING": 执行中, "FAILED": 执行失败, "SUCCESS": 执行成功, "INIT": 待执行 |

### 调用示例

#### 获取详细信息请求参数示例

```json
{
  "order_id":1001,
  "status":"FAILED"
}
```

### 响应示例

#### 获取详细信息返回结果示例

```json
{
  "result":true,
  "code":0,
  "message":"success",
  "data":{
    "info":[
        {
          "ip":"10.101.2.195"
        }
    ]
  }
}
```

### 响应参数说明

| 参数名称    | 参数类型       | 描述               |
|------------|--------------|--------------------|
| result     | bool         | 请求成功与否。true:请求成功；false请求失败 |
| code       | int          | 错误编码。 0表示success，>0表示失败错误  |
| message    | string       | 请求失败返回的错误信息 |
| data	     | object array | 响应数据             |

#### data

| 参数名称 | 参数类型       | 描述                |
|---------|--------------|---------------------|
| info    | object array | 回收主机列表信息       |

#### data.info

| 参数名称     | 参数类型 | 描述                 |
|------------|---------|----------------------|
| ip         | string  | 资源回收主机IP         |
